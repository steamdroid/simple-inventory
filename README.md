# Simple Inventory

Простой список наличия чего-либо, например, продуктов.
Vue3 + Bulma + иконки css.gg. Вдохновлен Apple Reminders.
Пока что умеет хранить данные только в localStorage.

## Сборка проекта

Установить зависимости:

```sh
npm install
```

Режим разработки

```sh
npm run dev
```

Сборка

```sh
npm run build
```
